# moneycrew

### Information

Discord bot running on JONTE's MONEY CREW.

https://discord.gg/tNqR3dG

### Features

- React-for-role channel access.
- [FACEIT](https://www.faceit.com/) cs:go integration (`!faceit`)
- Status commands (`!uptime`).
